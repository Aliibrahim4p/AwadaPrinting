## **Tech Stack**

- **Frontend:** React
- **Backend:** PHP (REST API)

- **Other:** Axios for API calls ...

## **Features**

- Manage purchases and stock inventory
- Build custom products from components
- Create sales orders and invoices
- Track order status and inventory items levels
- Track deliveries and reserved stock
- Client management and credits tracking
- Suppliers management and depts tracking
- Financial results showing all details
- Track labor,expenses,utilities
- Waste management
- 3rd party cots managemnets(ex tailoring,ironing...)
